・このファイルの中身について

　こちらは、自分が作成したゲームのプレイ動画となっており、1分程度の動画が入っております。
どちらも短めの内容で、完成度としてはあまり高くはありませんが、一度目を通してもらえると幸いです。

・TestMovement
このゲームは、2Dアクションゲームで、私が一番初めに作ったゲームとなります。方向キーで移動、スペースキーで
ジャンプを駆使し、ゴールの扉まで進めるのが目的です。
初めの作品だけあって、全体的に短めで、ところどころおかしな部分が多く。製作日数が大幅にかかってしまったの
が反省点となります。


・Unity-Chan! Figure
このゲームは、3Dゲームで、Unityでは有名なUnity-Chanのモデルを利用して作ったゲームとなります。Unity-Chanを
フィギュアに見立て、部屋の中で鑑賞することが出来る内容となっております。マウスで視点変更、マウスホイール
で距離の変更、右上のボタンでポーズの変更ができます。また、方向キーでも変更できます。
Unity-Chanのモデルを使用する際、初めから導入されていたソースコードにはほとんど改良をを加えず、一部の機能
は自力でソースコードを作って作成しました。
フリーの素材を駆使して使ったこともあり、要素がどうしても薄くなってしまったのが反省点となります。
